# stat386-spotify-project

A Python package and interactive Streamlit app for exploring Spotify music data. Using a dataset of approximately 170,000 songs collected through the Spotify API, this project provides tools to clean and prepare the data, analyze artist consistency and audio feature trends over time, and recommend similar songs using cosine similarity.

The `spotify_tools` package includes functions for data cleaning, summary statistics, feature analysis, and song recommendation. The Streamlit app makes these analyses accessible interactively, allowing users to explore any artist in the dataset or the full catalog across three key audio features: song length, acousticness, and tempo.
