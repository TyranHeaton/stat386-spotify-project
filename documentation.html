Data Cleaning Functions
def column_cleaning(df):
#Convert miliseconds into seconds and rename the column
    df = df[df['duration_ms'] > 53000]
    df['duration_ms'] = df['duration_ms']/1000
    df = df.rename(columns = {'duration_ms' : 'duration_s'})

    df['explicit'] = df['explicit'].map({1: 'Yes', 0: 'No'})
    return df

The Spotify API defaults to storing song length in milliseconds. This function converts that to seconds to make values smaller and easier to understand. Additionally, this function changes the 1s and 0s in the explicit column to Yes and No.

def reduce_list(df):
    df = df.groupby('artists').filter(lambda x: len(x) >= 40).reset_index(drop = True)
    df = df.groupby('artists').filter(lambda x: x['year'].nunique() >= 10)
    df = df.groupby('artists').filter(lambda x: x['acousticness'].mean() <= .75)
    return df

This function applies three different filters to the data. The first filter removes artists with fewer than 40 songs. The second removes artists with fewer than 10 unique years. The final filter removes highly acoustic artists, which often includes classical or instrumental catalogs. These filters help the data work better for individual artist analysis, especially in the app.

Analysis Functions
def artist_summary(df, artist):

    df = df[df['artists'] == artist]

    return df.describe()[['duration_s', 'energy', 'acousticness', 'tempo']]

This function uses the built-in describe() function to create a table that includes summary statistics for the listed variables.

def feature_trend(df, feature):

    df = df.copy()
    df[feature] = pd.to_numeric(df[feature], errors='coerce')

    return df.groupby('year')[feature].mean().reset_index()

This function is useful for analysis because it accepts a selected feature and returns each year with that feature’s average value. It is useful for seeing how specific aspects of music have changed over time.

def top_songs(df, feature, n=10):

    df = df.copy()

    # Check column exists
    if feature not in df.columns:
        raise ValueError(f"{feature} is not a valid column")

    # Convert to numeric safely
    df[feature] = pd.to_numeric(df[feature], errors='coerce')

    # Drop missing values
    df = df.dropna(subset=[feature])

    # Remove duplicates
    df = df.drop_duplicates(subset=['name', 'artists'])

    # Sort and select top n
    top = df.sort_values(feature, ascending=False).head(n)

    # Return clean output
    return top[['name', 'artists', feature]]

This function takes the selected feature and returns the top 10 songs ranked by that feature.

def song_variability(df):

    features = ['duration_s', 'energy', 'acousticness', 'tempo', 'valence']

    grouped = df.groupby('artists')[features].std()

    grouped['overall_variability'] = grouped.mean(axis=1)

    return grouped.sort_values('overall_variability', ascending=False)

This function is useful for measuring artist consistency. It takes five variables, calculates each artist’s standard deviation for those variables, and then computes overall variability as the mean of those standard deviations. This helps identify which artists are the most and least variable across these features.

def feature_correlations(df):

    features = ['duration_s', 'energy', 'acousticness', 'tempo', 'valence']

    return df[features].corr()

This function uses ‘duration_s’, ‘energy’, ‘acousticness’, ‘tempo’, and ‘valence’ and then applies the built-in Python correlation method to create a 5x5 matrix showing the relationship between features.

Recommendation Function
from sklearn.metrics.pairwise import cosine_similarity


FEATURE_COLS = [
 'energy', 'acousticness',
    'tempo', 'valence'
]
def recommend_similar(df, song_name, n=5):

    df = df.copy()

    # Convert to numeric
    for col in FEATURE_COLS:
        df[col] = pd.to_numeric(df[col], errors='coerce')

    df = df.dropna(subset=FEATURE_COLS).reset_index(drop=True)

    if song_name not in df['name'].values:
        raise ValueError("Song not found")

    # Feature matrix
    X = df[FEATURE_COLS].values

    # Get the selected song vector
    idx = df[df['name'] == song_name].index[0]
    song_vec = X[idx].reshape(1, -1)

    # 🔥 ONLY compute similarity to this one song
    sim_scores = cosine_similarity(song_vec, X).flatten()

    # Get top matches
    top_idx = sim_scores.argsort()[::-1][1:n+1]

    return df.iloc[top_idx][['name', 'artists']]

This function allows the user to see n songs that are most similar to the selected song. It uses the cosine_similarity() function from the sklearn package to measure similarity in a multidimensional feature space.
